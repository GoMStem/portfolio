# 블로그 글 작성 방법

이 글은 예시입니다. 앞으로 이런 형태로 글을 작성하면 됩니다.

## 마크다운 기본 문법

### 제목

```
# 큰 제목
## 중간 제목
### 작은 제목
```

### 텍스트 강조

**굵게** 쓰고 싶으면 `**텍스트**`, *기울임*은 `*텍스트*`

### 목록

- 항목 하나
- 항목 둘
- 항목 셋

### 인용구

> 기억에 남기고 싶은 말, 또는 참고한 내용

### 코드

인라인 코드는 `이렇게`, 코드 블록은 아래처럼:

```javascript
console.log('Hello, World!')
```
